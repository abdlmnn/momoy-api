## Prerequisites

Before running the commands, make sure you have:

- **Python** ‚Üí version 3.13.7 or later

```bash
  python --version
```

OR

```bash
  py --version
```

- Install and check version of **pip**

```bash
  python get-pip.py
```

```bash
  pip --version
```

---

## Virtual Environment

- Install virtual environtment

```bash
  pip install pipenv
```

- Set up virtual environtment

```bash
  pipenv shell
```

## Requirements

- Crate a file **requirements.txt** and copy and paste it

```php
django
djangorestframework
django-allauth
environs
pillow
python-dotenv
requests
cryptography
social-auth-app-django
pipenv
django-cors-headers
python-decouple
psycopg2-binary
```

- Install the file **requirements.txt**

```bash
  pip install -r requirements.txt
```

- Optional if you want to verify installation

```bash
  pip list
```
